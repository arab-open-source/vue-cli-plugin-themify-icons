
## How to use the component

### 1. import the `ThemifyIcon` component from `"vue-themify-icons"`

### 2. use it

```vue
<template>
  <div>
    <ThemifyIcon icon="user" />
  </div>
</template>

<script>
import ThemifyIcon from "vue-themify-icons";

export default {
  components: {
    ThemifyIcon
  }
};
</script>

```
