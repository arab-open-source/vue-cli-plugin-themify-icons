# vue-themify-icons

light modern Themify Icons for [Vue.js](https://github.com/vuejs/vue) 

## Installation

```
yarn add vue-themify-icons
// or npm install -S vue-themify-icons
```

## Usage

```vue
<template>
  <div>
    <ThemifyIcon icon="user" />
  </div>
</template>

<script>
import ThemifyIcon from "vue-themify-icons";

export default {
  components: {
    ThemifyIcon
  }
};
</script>

```
